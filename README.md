## 丨简介
[LittlePaimon](https://github.com/CMHopeSunshine/LittlePaimon.git) 的部署脚本，帮助小白搭建自己的小派蒙。

## 丨使用教程
1.下载脚本
```
git clone https://github.com/Hiroshi12138/LittlepaimonFastDeployment.git
```
2.将安装脚本移动到合适位置
```
cp 'LittlepaimonFastDeployment/ubuntu.sh' ubuntu.sh
```
3.运行脚本
```
bash ubuntu.sh
```
4.等待脚本运行结束
